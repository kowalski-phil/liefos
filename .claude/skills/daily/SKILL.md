If it doesn't exist, create it with this template:

# [Today's Date]

## Today's Focus

## Notes

If a journal entry already exists for today (same file), read it for context — the journal and daily note share the same file.

Then:
1. Check inbox/ for any unprocessed files and list them
2. Read the most recent 3 daily/ entries for context and momentum
3. Check projects/ for active work
4. Summarize the top 3 priorities based on recent notes
5. Ask: "What are we working on today?"
