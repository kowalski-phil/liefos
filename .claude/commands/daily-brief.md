## Important

- Be concise. This should take 3-5 minutes to read, not 15.
- Every item should include **why it matters to Philipp specifically** — not just generic news.
- Use web search to find actual current news. Don't make things up or use stale information.
- If a topic area has no noteworthy news today, skip it rather than padding with weak items.
- Link to sources where possible.
