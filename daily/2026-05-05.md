## Claude Code & Lola — The Architecture of Behavior Change

Set up Claude Code (Open Claw-style AI assistant, but more sophisticated and safer) a couple of days ago. Didn't see the benefit at first — another tool, nice but why? Now I'm getting the hang of it. Set up an email triage agent, connected it to Telegram for voice communication. It works really well. The vision: a mission control center with multiple agents that can discuss with each other and with me, delegating different tasks. Want this up and running before the Berlin community meeting end of May.

The most interesting setup: Lola (the bot named herself, I accepted). I configured her so that whenever I use inappropriate language — swear words, nastiness — she doesn't stop talking, she just reminds me it's not okay. The setup alone changed my behavior. I know how she'll react, so I don't use the language in the first place.

That's the same principle as these journals. I know Claude will ask about hydration, standing, exercise — so I drink, I stand up, I exercise. The accountability trick. Do I know it's a trick? Yes. Does knowing make it less effective? Not at all. You build the guardrail, you walk up to it every day, and the behavior follows. That's not self-deception. That's architecture.
