- Multi-bank cash concentration is common in organizations with distributed operations or regulatory requirements across jurisdictions
- Manual reconciliation of multi-bank positions is consistently cited as a top-3 pain point in treasury operations surveys

**Join Type Reference:**
- Inner Join: Returns only rows where the join key exists in both input tables
- Left Outer Join: Returns all rows from left table, matching rows from right table (nulls where no match)
- Right Outer Join: Returns all rows from right table, matching rows from left table (nulls where no match)
- Full Outer Join: Returns all rows from both tables, with nulls where no match exists
- For Tina's use case (accounts must exist in all banks to be reportable), Inner Join is the correct choice
