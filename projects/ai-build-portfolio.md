## How to Use This Portfolio

### For the AI Expert Network profile:
Pick 3-5 strongest projects. Lead with the problem, not the tech. Example: "Built a presentation analysis tool that extracts structured insights from any PowerPoint file — in one afternoon."

### For LinkedIn posts:
Each project is a potential story. Format: problem → what I built → how long it took → result.

### For client pitches:
Group by use case: content automation, data analysis, learning/training, strategic planning. Show range.

### For speaking engagements:
The sheer volume is the story. "I've built 20+ working tools with AI in the last 6 months. Here's what I learned."
